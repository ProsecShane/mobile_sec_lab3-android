### Использованные технологии

Технологии разработки Android приложений, задействованные в проекте:
* IDE платформа и язык программирования
    * Android Studio
    * Kotlin
* Архитектура
    * MVVM
    * Coroutine и Flow
* DI
    * DaggerHilt
* UI
    * View
* Бэкенд
    * ipinfo.io
* Network
    * Retrofit
* Хранение данных
    * Room
